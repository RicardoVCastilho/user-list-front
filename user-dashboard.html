Welcome to Userlist Dashboard, Stranger!

Este projeto consiste em um layout frontend integrado a uma API de autenticação de usuários com JWT. Ele permite realizar login e cadastro através dos métodos POST /login e POST /register. Além disso, é possível executar as operações básicas de CRUD em uma lista de usuários cadastrados, oferecendo uma interface intuitiva para gerenciar quem tem acesso ao sistema.
